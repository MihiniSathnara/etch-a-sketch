# Etch-a-Sketch

A dynamic, browser-based Etch-a-Sketch application that lets users create and interact with a customizable drawing grid.

## Features

- Hover over cells to progressively darken their color from light to fully black.
- Users can resize the grid to any size between 1x1 and 100x100.
- Squares automatically adjust their size to fit the container.
- Built entirely using HTML, CSS, and JavaScript. No external libraries required.
